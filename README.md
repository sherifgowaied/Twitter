# Twitter
the app supports:</br>
* Share posts * Like and dislike * Edit and delete your own posts * Profiles  * Follow and unfollow * Direct messages, seen , and mark as unseen * Pagination 
* Login system and Register * comments(in backend only "data base") * admin pannel * Timestamps about shared posts and messages * Notifaction
